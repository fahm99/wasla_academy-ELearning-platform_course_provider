## كيفية الاستخدام

### رفع صورة للكورس:
1. افتح نافذة إضافة/تعديل كورس
2. اضغط على منطقة رفع الصورة
3. اختر صورة (JPG, PNG)
4. معاينة الصورة
5. احفظ الكورس (سيتم رفع الصورة تلقائياً)

### رفع درس فيديو:
1. افتح نافذة إضافة درس
2. اختر نوع "فيديو"
3. اضغط "اختر ملف فيديو"
4. اختر ملف فيديو
5. سيتم تلقائياً:
   - استخراج اسم الفيديو → ملء حقل العنوان
   - استخراج مدة الفيديو → ملء حقل المدة
   - ملء حقل الوصف باسم الفيديو
6. اضغط "حفظ"
7. سيتم:
   - ضغط الفيديو (إذا كان الحجم كبير)
   - عرض مؤشر التقدم
   - رفع الفيديو
   - حفظ الدرس

## مؤشر التقدم

### الشكل:
```
┌─────────────────────────────┐
│  🔄 جاري رفع الفيديو...    │
│                             │
│     ⭕ 45%                  │
│  ████████░░░░░░░░░░░        │
│                             │
│  الحجم: 12.5 MB             │
│  المدة: 15 دقيقة            │
└─────────────────────────────┘
```

### المراحل:
1. **0-10%**: بدء العملية
2. **10-30%**: ضغط الفيديو (إذا لزم)
3. **30-90%**: رفع الملف
4. **90-100%**: إنهاء العملية

---

## ضغط الفيديو

### متى يتم الضغط؟
- إذا كان حجم الفيديو > 50 MB
- تلقائياً قبل الرفع

### إعدادات الضغط:
```dart
VideoQuality.MediumQuality  // جودة متوسطة
frameRate: 30               // 30 إطار/ثانية
```

### النتائج المتوقعة:
- تقليل الحجم: 50-70%
- الجودة: مقبولة للتعليم الإلكتروني
- السرعة: أسرع في الرفع

## استخراج معلومات الفيديو

### المعلومات المستخرجة:
1. **الاسم**: من اسم الملف
2. **المدة**: بالثواني (يتم تحويلها للدقائق)
3. **الحجم**: بالميجابايت

### الملء التلقائي:
```dart
// العنوان
_titleController.text = videoName;

// الوصف
_descriptionController.text = videoName;

// المدة (بالدقائق)
_videoDurationController.text = durationInMinutes.toString();
```

## الأخطاء الشائعة وحلولها

### 1. فشل ضغط الفيديو
**السبب**: مكتبة video_compress غير مثبتة
**الحل**:
```bash
flutter pub get
flutter clean
flutter pub get
```

### 2. لا يتم استخراج مدة الفيديو
**السبب**: صيغة الفيديو غير مدعومة
**الحل**: استخدم MP4 أو MOV

### 3. مؤشر التقدم لا يتحرك
**السبب**: محاكاة التقدم فقط (Supabase لا يدعم progress callbacks)
**الحل**: هذا طبيعي، المؤشر يعمل بشكل تقريبي

## ملاحظات مهمة

### للويب:
- ضغط الفيديو قد لا يعمل (قيود المتصفح)
- استخراج المعلومات محدود
- يُنصح باستخدام ملفات مضغوطة مسبقاً

### للموبايل/ديسكتوب:
- جميع الميزات تعمل بشكل كامل
- ضغط الفيديو فعال
- استخراج المعلومات دقيق

### الأداء:
- ضغط فيديو 100 MB قد يستغرق 1-3 دقائق
- الرفع يعتمد على سرعة الإنترنت
- يُنصح بإظهار رسالة انتظار للمستخدم
