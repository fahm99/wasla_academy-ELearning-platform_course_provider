## 📊 ملخص سريع

### الأدوار الثلاثة:

```
🎓 الطالب
├── التسجيل والمصادقة
├── تصفح الكورسات
├── الاشتراك (مجاني/مدفوع)
├── مشاهدة الدروس
├── حل الامتحانات
├── الحصول على الشهادات
└── إدارة الملف الشخصي

🧑‍🏫 مقدم الخدمة
├── التسجيل والمصادقة
├── إنشاء الكورسات
├── إدارة المحتوى
├── إنشاء الامتحانات
├── متابعة الطلاب
├── إدارة الشهادات
├── إدارة المدفوعات
└── إدارة الملف الشخصي

🛡️ الإدمن
├── إدارة المستخدمين
├── مراجعة مقدمي الخدمات
├── مراجعة الكورسات
├── مراقبة المدفوعات
├── إدارة الشهادات
├── إرسال الإشعارات
├── عرض التحليلات
└── إدارة الإعدادات
```

```
6. enrollments (التسجيل)
7. lesson_progress (تقدم الدروس)
8. exams (الامتحانات)
9. exam_questions (أسئلة الامتحانات)
10. exam_results (نتائج الامتحانات)
11. certificates (الشهادات)
12. payments (المدفوعات)
13. notifications (الإشعارات)
14. reviews (التقييمات)
15. app_settings (الإعدادات)
```

## 🔐 الأمان

جميع الجداول محمية بـ **Row Level Security (RLS)**:

- **الطلاب**: يرون فقط بياناتهم الخاصة
- **مقدمو الخدمات**: يرون فقط كورساتهم وطلابهم
- **الإدمن**: يرون كل البيانات
