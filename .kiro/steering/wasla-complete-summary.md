## 📊 هيكل قاعدة البيانات

### الجداول الأساسية:

```
1. users (المستخدمون)
   ├── student_fields (حقول الطالب)
   ├── provider_fields (حقول مقدم الخدمة)
   └── admin_fields (حقول الإدمن)

2. courses (الكورسات)
   ├── modules (الوحدات)
   │   └── lessons (الدروس)
   │       └── lesson_resources (الموارد)
   │       └── lesson_progress (التقدم)
   │
   ├── exams (الامتحانات)
   │   ├── exam_questions (الأسئلة)
   │   └── exam_results (النتائج)
   │
   ├── enrollments (التسجيل)
   ├── certificates (الشهادات)
   ├── payments (المدفوعات)
   └── reviews (التقييمات)

3. notifications (الإشعارات)
4. app_settings (الإعدادات)
```

### العلاقات:

```
users (1) ──→ (N) courses (مقدم الخدمة)
users (1) ──→ (N) enrollments (طالب)
courses (1) ──→ (N) modules
modules (1) ──→ (N) lessons
lessons (1) ──→ (N) lesson_resources
lessons (1) ──→ (N) lesson_progress
courses (1) ──→ (N) exams
exams (1) ──→ (N) exam_questions
exams (1) ──→ (N) exam_results
courses (1) ──→ (N) certificates
courses (1) ──→ (N) payments
courses (1) ──→ (N) reviews
```

## 🔐 سياسات الأمان (RLS)

### للطلاب:
```
✅ يرى الكورسات المنشورة فقط
✅ يرى بيانات التسجيل الخاصة به فقط
✅ يرى نتائجه الخاصة فقط
✅ يرى تقدمه الخاص فقط
✅ يرى شهاداته الخاصة فقط
```

### لمقدمي الخدمات:
```
✅ يرى كورساته الخاصة فقط
✅ يرى طلاب كورساته فقط
✅ يرى نتائج امتحانات كورساته فقط
✅ يمكنه تعديل كورساته فقط
```

### للإدمن:
```
✅ يرى كل البيانات
✅ يمكنه تعديل أي بيانات
✅ يمكنه حذف أي بيانات
```

## 🛠️ الخدمات المطلوبة

### 1. SupabaseService (الخدمة الرئيسية)
- التهيئة والاتصال

- التسجيل والمصادقة
- استعادة كلمة المرور

### 3. CourseService
- إدارة الكورسات (CRUD)
- البحث والفلترة
- إدارة الحالات

### 4. EnrollmentService
- الاشتراك في الكورسات
- تتبع التقدم
- إدارة الحالات

### 5. ExamService
- إنشاء الامتحانات
- إضافة الأسئلة
- تصحيح الإجابات

### 6. CertificateService
- تخصيص القوالب
- إصدار الشهادات
- التحقق والإلغاء

### 7. PaymentService
- معالجة الدفع
- إدارة المدفوعات
- الاسترجاع

### 8. StorageService
- رفع الفيديوهات
- رفع الملفات
- إدارة التخزين

### 9. NotificationService
- إرسال الإشعارات
- إدارة التفضيلات
