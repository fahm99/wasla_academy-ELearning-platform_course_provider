# 🎓 User Stories - الطالب (Student)

## 🎯 الهدف
التعلم، شراء الكورسات، إكمالها، والحصول على شهادة.

---

## 🟢 الحساب والمصادقة

### US-S-001: إنشاء حساب جديد
**كطالب** أريد إنشاء حساب حتى أتمكن من استخدام المنصة

**المتطلبات:**
- إدخال البريد الإلكتروني والكلمة المرورية
- التحقق من البريد الإلكتروني
- حفظ البيانات الأساسية (الاسم، الهاتف)
- إعادة توجيه للصفحة الرئيسية

**الحقول المطلوبة:**
```
- email (فريد)
- password (آمن)
- name
- phone (اختياري)
```

---

### US-S-002: تسجيل الدخول
**كطالب** أريد تسجيل الدخول حتى أصل إلى كورساتي

**المتطلبات:**
- إدخال البريد والكلمة المرورية
- التحقق من البيانات
- حفظ جلسة العمل
- إعادة توجيه للصفحة الرئيسية

---

### US-S-003: استعادة كلمة المرور
**كطالب** أريد استعادة كلمة المرور في حال نسيتها

**المتطلبات:**
- إدخال البريد الإلكتروني
- إرسال رابط إعادة تعيين
- تعيين كلمة مرور جديدة
- تأكيد التغيير

---

## 🟢 تصفح الكورسات

### US-S-004: عرض قائمة الكورسات
**كطالب** أريد تصفح الكورسات حتى أجد ما يناسبني

**المتطلبات:**
- عرض جميع الكورسات المنشورة
- عرض الصورة المصغرة والعنوان والسعر
- عرض تقييم الكورس
- عرض عدد الطلاب المسجلين
- ترقيم الصفحات

**البيانات المعروضة:**
```
- thumbnail_url
- title
- price
- rating
- students_count
- provider_name
```

---

### US-S-005: البحث عن كورس
**كطالب** أريد البحث عن كورس باسم أو موضوع معين

**المتطلبات:**
- حقل بحث نصي
- البحث في العنوان والوصف
- عرض النتائج فوراً
- تصفية حسب الفئة والمستوى

**معايير البحث:**
```
- title (يحتوي على)
- category
- level
- price_range
- rating
```

---

### US-S-006: عرض تفاصيل الكورس
**كطالب** أريد مشاهدة تفاصيل الكورس قبل الاشتراك

**المتطلبات:**
- عرض الوصف الكامل
- عرض الوحدات والدروس
- عرض معلومات المدرب
- عرض التقييمات والتعليقات
- عرض السعر والعروض

**البيانات المعروضة:**
```
- description
- modules (عدد)
- lessons (عدد)
- provider_info
- reviews
- price
- duration_hours
```

## 🟢 الاشتراك والدفع

### US-S-007: الاشتراك في كورس مجاني
**كطالب** أريد الاشتراك في كورس مجاني مباشرة

**المتطلبات:**
- زر اشتراك مباشر
- تأكيد الاشتراك
- إضافة الكورس للقائمة الخاصة بي
- إعادة توجيه للكورس

**العملية:**
```
1. عرض زر "اشترك الآن"
2. تأكيد الاشتراك
3. إنشاء enrollment record
4. إعادة توجيه للكورس
```

---

### US-S-008: عرض معلومات الدفع للكورس المدفوع
**كطالب** أريد رؤية معلومات الدفع الخاصة بمقدم الخدمة

**المتطلبات:**
- عرض سعر الكورس
- عرض معلومات الدفع (محفظة/بنك)
- عرض التعليمات الإضافية
- نسخ معلومات الدفع بسهولة

**البيانات المعروضة:**
```
- course_price (سعر الكورس)
- currency (العملة)
- wallet_number (رقم المحفظة)
- wallet_owner_name (اسم صاحب المحفظة)
- bank_name (اسم البنك)
- bank_account_number (رقم الحساب)
- iban (رقم الآيبان)
- additional_info (تعليمات إضافية)
```

---

### US-S-009: إجراء الدفع ورفع الإيصال
**كطالب** أريد إجراء الدفع ورفع إيصال التحويل

**المتطلبات:**
- إجراء التحويل البنكي أو الدفع عبر المحفظة
- رفع صورة الإيصال
- إدخال رقم العملية
- إدخال معلومات إضافية
- إرسال طلب الدفع

**الحقول المطلوبة:**
```
- receipt_image (صورة الإيصال) *
- transaction_reference (رقم العملية) *
- payment_method (طريقة الدفع) *
- amount (المبلغ المدفوع) *
- payment_date (تاريخ الدفع)
```

**طرق الدفع المدعومة:**
```
- bank_transfer (تحويل بنكي)
- local_transfer (تحويل محلي/محفظة)
```

---

### US-S-010: متابعة حالة طلب الدفع
**كطالب** أريد متابعة حالة طلب الدفع الخاص بي

**المتطلبات:**
- عرض حالة الطلب الحالية
- عرض تاريخ الإرسال
- استقبال إشعار عند التحديث
- رؤية سبب الرفض (إن وجد)

**حالات الدفع:**
```
- pending (قيد المراجعة)
  - الوصف: تم إرسال الطلب وينتظر مراجعة المقدم
  - الإجراء: انتظر المراجعة
  
- completed (مكتمل)
  - الوصف: تم قبول الدفع وتفعيل الوصول للكورس
  - الإجراء: يمكنك الآن الوصول للكورس
  
- failed (مرفوض)
  - الوصف: تم رفض الدفع
  - الإجراء: راجع السبب وأعد المحاولة
  
- refunded (مسترد)
  - الوصف: تم استرداد المبلغ
  - الإجراء: تم إلغاء التسجيل
```

---

### US-S-011: إعادة إرسال طلب دفع مرفوض
**كطالب** أريد إعادة إرسال طلب الدفع إذا تم رفضه

**المتطلبات:**
- قراءة سبب الرفض
- تصحيح المعلومات
- رفع إيصال جديد
- إعادة إرسال الطلب

**البيانات المعروضة:**
```
- rejection_reason (سبب الرفض)
- rejected_at (تاريخ الرفض)
- verified_by (المراجع)
```

---

### US-S-012: الوصول للكورس بعد قبول الدفع
**كطالب** أريد الوصول للكورس مباشرة بعد قبول الدفع

**المتطلبات:**
- استقبال إشعار بقبول الدفع
- تفعيل الوصول تلقائياً
- إنشاء enrollment record
- إعادة توجيه للكورس

**الإجراءات التلقائية:**
```
- تحديث payment_status إلى "completed"
- تحديث enrollment_status إلى "active"
- تحديث payment_status في enrollments إلى "paid"
- إرسال إشعار للطالب
- إضافة الكورس لقائمة "كورساتي"
```

## 💳 نظام الدفع المحلي - دليل الطالب

### تدفق عملية الدفع للطالب:

**الخطوة 1: اختيار الكورس**
- تصفح الكورسات المتاحة
- عرض تفاصيل الكورس والسعر
- الضغط على "اشترك الآن"

**الخطوة 2: عرض معلومات الدفع**
- عرض سعر الكورس
- عرض معلومات الدفع الخاصة بالمقدم:
  - رقم المحفظة الإلكترونية
  - معلومات الحساب البنكي
  - رقم الآيبان
  - تعليمات إضافية
- نسخ المعلومات بسهولة

**الخطوة 3: إجراء الدفع**
- تحويل المبلغ عبر:
  - المحفظة الإلكترونية (STC Pay, مدى Pay, إلخ)
  - التحويل البنكي
- الاحتفاظ بإيصال الدفع

**الخطوة 4: رفع الإيصال**
- رفع صورة الإيصال
- إدخال رقم العملية
- اختيار طريقة الدفع
- تأكيد المبلغ المدفوع
- إرسال الطلب

**الخطوة 5: انتظار المراجعة**
- حالة الطلب: "قيد المراجعة"
- استقبال إشعار عند المراجعة
- متابعة الحالة في قسم "مدفوعاتي"

**الخطوة 6: النتيجة**

**إذا تم القبول:**
- استقبال إشعار بالقبول
- تفعيل الوصول للكورس تلقائياً
- البدء في التعلم مباشرة

**إذا تم الرفض:**
- استقبال إشعار بالرفض مع السبب
- مراجعة السبب
- تصحيح المعلومات
- إعادة إرسال الطلب

### معلومات مهمة:

**نصائح للدفع الناجح:**
```
✓ تأكد من إدخال المبلغ الصحيح
✓ احتفظ بإيصال واضح وقابل للقراءة
✓ أدخل رقم العملية بدقة
✓ اتبع التعليمات الإضافية من المقدم
✓ تأكد من اسم المستلم قبل التحويل
```

**أسباب الرفض الشائعة:**
```
✗ إيصال غير واضح
✗ مبلغ غير صحيح
✗ رقم عملية خاطئ
✗ تحويل لحساب خاطئ
✗ معلومات ناقصة
```

**وقت المراجعة:**
```
- عادة: 24-48 ساعة
- يعتمد على مقدم الخدمة
- ستصلك إشعارات فورية
```

### حالات الدفع بالتفصيل:

| الحالة | الأيقونة | الوصف | ماذا تفعل؟ |
|-------|---------|-------|-----------|
| **pending** | ⏳ | قيد المراجعة | انتظر المراجعة |
| **completed** | ✅ | مكتمل | ابدأ التعلم |
| **failed** | ❌ | مرفوض | راجع السبب وأعد المحاولة |
| **refunded** | 💰 | مسترد | تم استرداد المبلغ |
