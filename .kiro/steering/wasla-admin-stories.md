# 🛡️ User Stories - الإدمن (Admin)

## 🎯 الهدف
إدارة المنصة بالكامل وضمان الجودة والأمان.

---

## 🟢 إدارة المستخدمين

### US-A-001: عرض جميع المستخدمين
**كأدمن** أريد عرض جميع المستخدمين

**المتطلبات:**
- عرض قائمة شاملة
- عرض معلومات المستخدم
- البحث والفلترة
- ترتيب حسب معايير مختلفة

**البيانات:**
```
- user_id
- name
- email
- user_type (student/provider/admin)
- is_active
- created_at
- last_login
```

---

### US-A-002: تفعيل/تعطيل الحسابات
**كأدمن** أريد تفعيل/تعطيل الحسابات

**المتطلبات:**
- اختيار المستخدم
- تغيير حالة الحساب
- إرسال إشعار
- تسجيل الإجراء

**الحالات:**
```
- active (مفعل)
- inactive (معطل)
- suspended (موقوف)
- deleted (محذوف)
```

---

### US-A-003: حذف المستخدمين المخالفين
**كأدمن** أريد حذف المستخدمين المخالفين

**المتطلبات:**
- تأكيد الحذف
- حذف جميع البيانات المرتبطة
- تسجيل السبب
- إرسال إشعار

---

### US-A-004: عرض سجل النشاط
**كأدمن** أريد عرض سجل نشاط المستخدم

**المتطلبات:**
- عرض جميع الأنشطة
- تصفية حسب النوع
- تصفية حسب التاريخ
- تصدير التقارير

**أنواع الأنشطة:**
```
- login
- logout
- course_enrollment
- payment
- certificate_issued
- content_uploaded
```
