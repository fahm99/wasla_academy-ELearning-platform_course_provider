# 🎯 دليل سريع: نظام الامتحانات

## 📍 كيف أصل للامتحانات؟

### الطريقة الأولى (الأسهل):
```
الكورسات 📚
    ↓
اختر كورس 📖
    ↓
زر "إدارة الامتحانات" 📝
```

### الطريقة الثانية:
```
الكورسات 📚
    ↓
اختر كورس 📖
    ↓
"إدارة المحتوى" 📂
    ↓
أيقونة الامتحانات 🎓 (أعلى يمين)
```

---

## ➕ كيف أضيف امتحان؟

### الخطوة 1: إنشاء الامتحان
```
اضغط زر "إضافة امتحان" (الزر الأزرق العائم)
    ↓
املأ البيانات:
  • العنوان: "امتحان الوحدة الأولى" ✏️
  • الوصف: "يغطي الدروس 1-5" (اختياري)
  • عدد الأسئلة: 10 (اختياري)
  • درجة النجاح: 60% 🎯
  • المدة: 30 دقيقة ⏱️
  • السماح بالإعادة: نعم ✅
  • عدد المحاولات: 3 🔄
    ↓
اضغط "إضافة"
```

### الخطوة 2: إضافة الأسئلة
```
اضغط على الامتحان الذي أنشأته
    ↓
اضغط زر "إضافة سؤال" (الزر الأزرق العائم)
    ↓
املأ السؤال:
  • نص السؤال: "ما هي عاصمة السعودية؟"
  • الخيار A: الرياض ← حدد هذا (الإجابة الصحيحة) ✅
  • الخيار B: جدة
  • الخيار C: الدمام
  • الخيار D: مكة
  • النقاط: 1
    ↓
اضغط "إضافة"
    ↓
كرر لإضافة المزيد من الأسئلة
```

### الخطوة 3: نشر الامتحان
```
ارجع لشاشة "إدارة الامتحانات"
    ↓
اضغط زر "نشر" 📤 في بطاقة الامتحان
    ↓
الامتحان الآن متاح للطلاب! 🎉
```

## 🎨 شرح الواجهة

### بطاقة الامتحان تحتوي على:
```
┌─────────────────────────────────────┐
│ امتحان الوحدة الأولى        [مسودة] │
│ يغطي الدروس من 1 إلى 5              │
│                                     │
│ 📝 10 أسئلة  ⏱️ 30 دقيقة          │
│ 🎯 النجاح: 60%  🔄 يسمح بالإعادة   │
│                                     │
│ [📤 نشر] [✏️ تعديل] [🗑️ حذف]      │
└─────────────────────────────────────┘
```

### بطاقة السؤال تحتوي على:
```
┌─────────────────────────────────────┐
│ [1] ما هي عاصمة السعودية؟          │
│                                     │
│ ⭕ A: الرياض ✅ (الإجابة الصحيحة)  │
│ ⭕ B: جدة                           │
│ ⭕ C: الدمام                        │
│ ⭕ D: مكة المكرمة                   │
│                                     │
│ النقاط: 1                           │
│ [✏️ تعديل] [🗑️ حذف]                │
└─────────────────────────────────────┘
```

## ⚡ نصائح سريعة

✅ **افعل:**
- راجع الأسئلة قبل النشر
- اختر وقتاً مناسباً (دقيقة لكل سؤال)
- فعّل خيار الإعادة للتعلم
- استخدم عناوين واضحة

❌ **لا تفعل:**
- لا تنشر امتحاناً بدون أسئلة
- لا تنسَ مراجعة الإجابات الصحيحة
- لا تجعل الوقت قصيراً جداً
- لا تنسَ حفظ التعديلات

## 🔍 أين أجد؟

| الميزة | المكان |
|--------|--------|
| إضافة امتحان | شاشة إدارة الامتحانات → زر ➕ |
| إضافة سؤال | تفاصيل الامتحان → زر ➕ |
| تعديل امتحان | بطاقة الامتحان → زر ✏️ |
| تعديل سؤال | بطاقة السؤال → زر ✏️ |
| نشر امتحان | بطاقة الامتحان → زر 📤 |
| عرض النتائج | شريط الأدوات → أيقونة 📊 |

## 🎓 مثال سريع

```
1. اذهب للكورسات
2. اختر "مقدمة في البرمجة"
3. اضغط "إدارة الامتحانات"
4. اضغط "➕ إضافة امتحان"
5. املأ: "امتحان الوحدة الأولى" - 10 أسئلة - 15 دقيقة
6. اضغط "إضافة"
7. اضغط على الامتحان
8. اضغط "➕ إضافة سؤال"
9. اكتب السؤال والخيارات
10. حدد الإجابة الصحيحة
11. اضغط "إضافة"
12. كرر للأسئلة الأخرى
13. ارجع واضغط "📤 نشر"
14. تم! 🎉
```
